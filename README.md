# Ubots Messages App

## Before all

Create a *.env* file and set the a **VITE_APU_URL**. This refer to a server app ***${host}:${port}***.

Ex:
```bash
VITE_API_URL=http://192.168.1.103:3000
```
 
## Install

```bash
pnpm i
```

## Running

```bash
pnpm dev
```

## Running Tests

```bash
npm vitest
```
